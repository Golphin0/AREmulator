## AREmulator -- a webtool for simulating the Nintendo 3DS AR Cards
This project is a tool to simulate the Nintendo 3DS AR Cards, for easy use in the AR Games application for the 3DS.\
This tool is not complete, and is not responsible for any unintended behavior (If even possible).
